Initializing the Hush Encryption Engine in Java
Detailed information on using the Hush Encryption Engine is available in the included API documentation. This document is a quick start guide intended to show you how to use the Hush Encryption Engine in a basic way. Once you get an idea of what you need to do, please read the API documentation for detailed information.

Using the Hush Encryption Engine in Java

The com.hush.hee.HushEncryptionEngineCore object provides core functionality for the Hush Encryption Engine. You can instantiate and initialize the Hush Encryption Engine in Java as follows: 

import java.util.Vector;

import com.hush.hee.HushEncryptionEngineCore;


public class Encrypt
{
		public static void main(String argv[]) throws Exception
		{
			String input = "My original text";
			
			HushEncryptionEngineCore hee = new HushEncryptionEngineCore();
			Vector recipients = new Vector();
			recipients.addElement("myhushmailaccountaddresshere@hushmail.com");
			
			// Makes the output be a string
			hee.useArmor(true);
			
			byte[] encrypted = hee.encrypt(input.getBytes("UTF-8"), recipients, null);
			
			String encryptedString = new String(encrypted, "UTF-8");
			
			System.out.println(encryptedString);
		}
}
